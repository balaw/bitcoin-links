Melainkan mengambil source code yang yang sama berasal dari <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Bitcoin&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Bitcoin diperkenalkan pertama kali oleh &amp;lt;strong&amp;gt;Satoshi Nakamoto&amp;lt;/strong&amp;gt;dalam makalah semi formal (disebut &amp;lt;strong&amp;gt;Whitepaper&amp;lt;/strong&amp;gt;)pada tahun 2008 sebagai &amp;lt;strong&amp;gt;&amp;quot;Sistem Pembayaran Tunai Elektronik Berbasis Peer-to-peer&amp;quot;&amp;lt;/strong&amp;gt;.&amp;lt;br data-rich-text-line-break=&amp;quot;true&amp;quot;/&amp;gt;&amp;lt;br data-rich-text-line-break=&amp;quot;true&amp;quot;/&amp;gt;Pengguna Bitcoin berasal dari latar belakang yang cukup beragam.Kenyataan itu membuat banyak orang memberikan penafsiran yang berbeda-beda tentang Bitcoin.Seperti halnya dengan menyebut Bitcoin sebagai sebuah &amp;lt;strong&amp;gt;mata uang berbasis kriptografi (cryptocurrency)&amp;lt;/strong&amp;gt;.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>bitcoin</span>, lalu diturunkan menjadi jenis lainnya lagi. Meski memang telah ada ribuan jenis <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Altcoin&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Sebuah sebutan untuk cryptocurrency alternatif selain Bitcoin.Litecoin,Feathercoin,PPcoin,Doge,dan yang lainnya,semuanya adalah Altcoins.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>altcoin</span>, nampaknya akan tetap ada bermunculan nama-nama kripto baru, dan terus bertambah. Orang di dunia nampak begitu menikmati betapa mudahnya membuat uang sendiri versi mereka, setelah ruang-ruang itu dimonopoli dan dimanipulasi oleh bank-bank sentral dunia. Mereka mentasbihkan dirinya sebagai dewa penguasa penciptaan mata uang.</p><p>Apa yang terjadi jika setiap orang di dunia membuat mata uangnya sendiri? Di mana posisi dan peran bank sentral nantinya? Kira-kira seperti itulah gambaran yang perlu menjadi prolog pada kesempatan kali ini. Daripada makin ngelantur :D, mari kita coba memulai bahasan tentang bagaimana cara membuat cryptocurrency sendiri.</p><p>Seperti yang telah dijelaskan di awal, bahwa dari ribuan jenis kripto yang ada, sebagian besar banyak mengambil dari source code yang sudah ada. Pada bahasan ini pula, kita akan memanfaatkan source code yang telah ada tersebut. Bagaimana pun, cara-cara seperti itu tetap menjadi titik awal yang baik untuk mencoba mengenali lebih mendalam tentang cryptocurrency, bitcoin secara khusus, karena telah menjadi pengawal dunia cryptocurrency.</p><p>Souce code yang kita ambil nantinya adalah <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Litecoin&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Sebuah digital currency alternatif selain bitcoin,dengan berbasis pada Scrypt proof of work.An altcoin based on the Scrypt proof of work.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>Litecoin</span> versi 0.8, pertimbangannya, karena masih simple, dan belum kompleks seperti yang saat ini. Bagi yang sudah pengalaman coding, mungkin referensi <a href="https://bitcoinmedia.id/pemprograman-blockchain/">pemprograman blockchain dengan C#</a> bisa berguna untuk lebih jauh bisa bereksplorasi.</p><p>Jangan diambil serius, karena ini hanya bersifat pembahasan saja, bukan bertujuan <a href="https://www.bitcoinmedia.id/cara-membuat-cryptocurrency-sendiri/">membuat kripto baru </a>seperti pada umumnya. Siapkan cemilan, kopi, karena prosesnya akan membutuhkan waktu yang lumayan panjang. Yuk, mari kita mulai.</p><h2 class="wp-block-heading "><strong>Bagian Satu &ndash; Install Semua Dependency yang Diperlukan</strong></h2><p class="has-drop-cap ">Dependencies ini adalah file-file yang diperlukan saat kita hendak menginstall sesuatu. Agar cara membuat cryptocurrency sendiri ini berjalan mulus, tentu membutuhkan sekian banyak dependency yang diperlukan. Langsung saja, buka terminal di ubuntu anda, lalu satu persatu paste perintah berikut ini:</p><pre class="wp-block-code "><code>sudo apt-get install git
sudo apt-get install build-essential libtool autotools-dev automake pkg-config libssl-dev libevent-dev bsdmainutils

sudo apt-get install libboost-all-dev
sudo apt-get install software-properties-common

sudo apt-get update
sudo apt-get install libdb4.8-dev libdb4.8++-dev
sudo apt-get install libminiupnpc-dev
sudo apt-get install libzmq3-dev
sudo apt-get install libqt5gui5 libqt5core5a libqt5dbus5 qttools5-dev qttools5-dev-tools libprotobuf-dev protobuf-compiler
sudo apt-get install libqt4-dev libprotobuf-dev protobuf-compiler</code></pre><p>Jika anda bertanya mengapa kita memerlukan repository bitcoin? Betul, karena Litecoin memang berasal dari sumber yang sama, Bitcoin. Lebih jauh kita akan mengetahui sendiri bagaimana Litecoin ini tidak akan jauh berbeda dengan Bitcoin, selain beberapa konsensus mendasar saja yang membedakan, seperti total supply dan beberapa hal lain.</p><p>Teruskan lagi prosesnya dengan memulai clone source code Litecoin versi 0.8 di github. Untuk memudahkan, sebaiknya anda menyimpan file clone Litecoin tersebut di folder Desktop pada ubuntu yang anda gunakan. Paste perintah di bawah ini:</p><pre class="wp-block-code "><code>cd Desktop
git clone -b 0.8 https://github.com/litecoin-project/litecoin.git</code></pre><p>Pertama kali setelah clone, ada baiknya anda merubah nama folder hasil clone tersebut, sehingga mudah untuk di ingat. Di sini, EB merubah nama foldernya dengan &ldquo;ebina&rdquo;. Jadi untuk selanjutnya, anda hanya tinggal menyesuaikan saja dengan folder anda sendiri.</p><p>Kalau sudah, lanjutkan dengan merubah semua nama Litecoin, menjadi apapun sesuka anda. Paste saja perintah dibawah ini:</p><pre class="wp-block-code "><code>cd ebina
find . -type f -print0 | xargs -0 sed -i 's/litecoin/ebina/g'
find . -type f -print0 | xargs -0 sed -i 's/Litecoin/Ebina/g'
find . -type f -print0 | xargs -0 sed -i 's/LiteCoin/EbinaCoin/g'
find . -type f -print0 | xargs -0 sed -i 's/LITECOIN/EBINA/g'
find . -type f -print0 | xargs -0 sed -i 's/LTC/INA/g'</code></pre><p>Jadi dengan perintah diatas tadi, kita telah merubah semua kata di dalam file-file hasil clone tersebut, dengan versi kita sendiri.</p><p>Khusus untuk source code Litecoin di versi ini, ada sedikit masalah di file <strong><em>rpcrawtransaction.cpp</em></strong>. Jadi kita perlu untuk merubahnya terlebih dulu. Buka folder hasil clone anda, lalu buka folder <em><strong>/src.</strong></em> Tepat di baris nomor 242, paste saja seperti ini:</p><pre class="wp-block-code "><code>const CScriptID&amp; hash = boost::get&lt;CScriptID&gt;(address);</code></pre><p>Setelah itu, sekarang kita coba untuk compile hasil clone. Jadi pastikan semua langkah diatas telah dilakukan secara benar.</p><p>Kalau berhasil, lebih jauh kita juga akan mengulang proses compile beberapa kali. Berhasil? Kalau berhasil berarti anda siap melangkah di tahap cara membuat cryptocurrency sendiri selanjutnya. Untuk mengetahui berhasil atau tidaknya, di dalam folder /src itu akan terdapat file <em><strong>&ldquo;ebinad&rdquo;</strong></em> <em>(jika di dalam folder /src anda <strong>&ldquo;xxxd&rdquo;</strong>)</em>. Fungsi file itu adalah sebagai daemon, kalau di bitcoin disebut dengan bitcoind <em>(daemon bitcoin)</em>.</p><h2 class="wp-block-heading "><strong>Bagian Dua, Cara Membuat Cryptocurrency &ndash; Mengatur Parameter Network</strong></h2><p class="has-drop-cap ">Mari kita lanjutkan tutorial cara membuat cryptocurrency ini. Bagian kedua sudah masuk pada hal yang cukup penting. Yakni dalam hal mengatur parameter network atau jaringan.</p><p>Kalau kita memperhatikan di bitcoin, ada dua port yang digunakan, yakni pada <strong>port 8332</strong> dan <strong>port 8333</strong>. Pada dasarnya, jenis komunikasi di dalam bitcoin dapat dilakukan dengan mudah melalui TCP saja, namun masih banyak ruang yang bisa dikembangkan untuk diluar itu.</p><p>Jika pada <strong><em>port 8333 lebih dominan dan ekslusif dipergunakan sebagai jalur komunikasi antar <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Node&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Satu titik sambungan dalam sebuah jaringan.Bisa diartikan berbagai&nbsp;perangkat/klien yang terhubung dalam sebuah jaringan.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>node</span> di dalam jaringannya</em></strong>, sedangkan <strong><em>port 8332 lebih bersifat inklusif karena klien itu haruslah mengaktifkan terlebih dahulu</em></strong> secara default untuk bisa menggunakannya.</p><p>Meski pada dasarnya perbedaan fungsional kedua port itu sama-sama bertindak sebagai <span class="glossaryLink " aria-describedby="tt " data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Peer-To-Peer&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Di singkat P2P,(ujung ke ujung).Adalah suatu sistem interaksi yang tidak hanya menghubungkan satu dengan yang lainnya,namun ujung-ujung ini saling berhubungan secara dinamis dan berpartisipasi dalam mengarahkan lalu lintas komunikasi informasi pemrosesan dan penugasan pembagian bandwith yang intensif.&lt;/div&gt;" data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>peer-to-peer</span> port karena memang tipikal jaringannya adalah peer-to-peer, namun banyak orang memang lebih banyak membedakan dari sisi fungsionalitasnya.Oleh karena itu, port 8333 bitcoin lebih bersifat sebagai peer-to-peer port untuk node, dan port 8332 lebih bersifat TCP port.</p><p>Sama juga dengan di Altcoin Litecoin, karena memang berasal dari akar yang sama, tidak banyak perbedaan. Port yang digunakan juga ada dua, yakni port 9333 dan 9332. Dalam hal ini, anda juga harus mengganti kedua port tersebut dengan port anda sendiri.</p><p class="has-very-light-gray-background-color has-background "><em><strong>Pertanyaannya, bagaimana kedua port tersebut bisa berfungsi secara maksimal tanpa bisa terganggu dan tercampur dengan jaringan yang lain?</strong> </em></p><p>Nah, dari bitcoin, yang kemudian digunakan juga oleh Litecoin, sama-sama menggunakan langkah ajaib untuk melindungi dan memastikan kedua port mereka tersebut dapat berlaku hanya untuk jaringan bitcoin dan litecoin saja. Bagaimana langkah ajaib tersebut bisa dilakukan? Anda akan mengetahuinya sendiri nantinya.</p><p>Kita tunda dulu penjelasan langkah ajaib proteksi network tadi, untuk selanjutnya langsung dengan mengubah port dengan port anda sendiri. Seperti yang dijelaskan tadi, bahwa port Litecoin adalah 9333 dan 9332, jadi kita akan mengganti kedua port itu dengan port anda sendiri. Pastikan anda berada di folder <em><strong>/ebina/</strong></em> (sesuaikan dengan folder anda), lalu paste sesuai port anda seperti berikut:</p><pre class="wp-block-code "><code>find . -type f -print0 | xargs -0 sed -i 's/9333/2333/g'
find . -type f -print0 | xargs -0 sed -i 's/9332/2332/g'</code></pre><h3 class="wp-block-heading "><strong>Menentukan Karakter Address</strong></h3><p>Berikutnya adalah mengganti karakter <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Address&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Adalah alamat bitcoin yang digunakan untuk mengirim dan menerima transaksi bitcoin.Alamat Bitcoin terdiri dari deret karakter huruf dan angka,dan bisa dimunculkan dalam QR Code.&nbsp;Alamat Bitcoin ini berfungsi juga sebagai public key (kunci publik),dalam sepasang kunci yang hanya dipegang oleh pemilik alamat Bitcoin untuk bisa memberikan tanda tangan digital dalam sebuah transaksi (digital sign).&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>Address</span> kripto yang bisa anda sesuaikan sendiri. Kalau di Bitcoin kita sudah banyak mengetahui karakter addressnya dimulai dengan angka <em><strong>&ldquo;1&rdquo;</strong></em>, sedangkan di Litecoin dengan karakter huruf <em><strong>&ldquo;L&rdquo;</strong></em> di depan. Karakter huruf <em><strong>&ldquo;L&rdquo;</strong></em> yang digunakan oleh Litecoin ini, diperoleh dari versi desimal angka <em><strong>&ldquo;48&rdquo;</strong></em>. Jadi nantinya, anda dapat leluasa menentukan karakter address anda sendiri.</p><p>Address kripto, adalah turunan dari <span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Hash&lt;/div&gt;&lt;div class=glossaryItemBody&gt;Dalam istilah komputer,hash berfungsi sebagai sebuah algoritma yang mengubah text menjadi sebuah sederetan karakter acak dan memiliki jumlah karakter yang sama.Sedangkan di dalam Bitcoin,hash/hashing merupakan hasil kompleks dari penghitungan matematika yang mudah untuk diproduksi ulang namun cukup sulit untuk mengulang,dan juga sulis untuk diprediksi.Sehingga proses sebuah hashing adalah proses matematika murni.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>hash</span> <a href="https://bitcoinmedia.id/public-key-sebagai-identitas/">public key </a>yang selanjutnya dirubah ke dalam format <a href="http://bitcoinmedia.id/bagaimana-menyimpan-bitcoin/"><em><strong>base58</strong></em></a>. Untuk merubahnya, buka file <em><strong><span class="glossaryLink "  aria-describedby="tt "  data-cmtooltip="&lt;div class=glossaryItemTitle&gt;Base58&lt;/div&gt;&lt;div class=glossaryItemBody&gt;&amp;lt;em&amp;gt;&amp;lt;strong&amp;gt;Base58&amp;lt;/strong&amp;gt;&amp;lt;/em&amp;gt;adalah satu set skema pengkodean teks binary yang digunakan untuk bisa mewakili besaran bilangan bulat sebagai teks alfanumerik.Contoh Base58 dalam Bitcoin:&amp;lt;em&amp;gt;&amp;lt;strong&amp;gt;123456789ABCDEFGHJKLMNPQRSTUVWXYZabcdefghijkmnopqrstuvwxyz&amp;lt;/strong&amp;gt;&amp;lt;/em&amp;gt;(berjumlah 58 karakter)Kalau kita perhatikan ada beberapa karakter yang tidak ada di Base58,seperti pada huruf kapital &amp;quot;&amp;lt;strong&amp;gt;O&amp;lt;/strong&amp;gt;&amp;quot;,huruf kapital i (&amp;quot;&amp;lt;strong&amp;gt;I&amp;lt;/strong&amp;gt;&amp;quot;),angka &amp;quot;&amp;lt;strong&amp;gt;0&amp;lt;/strong&amp;gt;&amp;quot;,dan huruf kecil dari L (&amp;quot;&amp;lt;strong&amp;gt;l&amp;lt;/strong&amp;gt;&amp;quot;).Perbedaan Base58 pada beberapa karakter tersebut yang membedakannya dengan &amp;lt;em&amp;gt;&amp;lt;strong&amp;gt;Base64&amp;lt;/strong&amp;gt;&amp;lt;/em&amp;gt;.Hal itu memang dimodifikasi sedemikian rupa untuk menghindari adanya kesalahan-kesalahan karena memang karakter tersebut cukup mirip.Sehingga pengguna pun bisa menuliskannya secara manual dengan lebih mudah.Ataupun cukup dengan copy paste saja keseluruhan string tersebut.&lt;/div&gt;"  data-gt-translate-attributes='[{"attribute":"data-cmtooltip", "format":"html "}]'>base58</span>.h</strong></em> yang ada di dalam folder <em><strong>/ebina/src</strong></em> melalui exploler ubuntu anda, lalu edit dengan text editor yang biasa anda gunakan.</p><p>Arahkan ke baris nomor 275 pada file tadi, dan ganti sesuai yang anda inginkan. Kurang lebihnya akan nampak seperti ini:</p><pre class="wp-block-code "><code>class CBitcoinAddress : public CBase58Data
{
public:
enum
{
PUBKEY_ADDRESS = 33, // Ebina addresses start with E
SCRIPT_ADDRESS = 5,
PUBKEY_ADDRESS_TEST = 92,
SCRIPT_ADDRESS_TEST = 196,
};</code></pre><p>Anda dapat melihat lebih jauh di <em><strong><a href="https://en.bitcoin.it/wiki/List_of_address_prefixes ">list prefix address</a></strong></em> untuk lebih memudahkan anda dalam menentukan angka desimal mana agar dapat merepresentasikan karakter huruf yang anda inginkan. Diatas, EB menggunakan angka desimal <em><strong>&ldquo;33&rdquo;</strong></em> untuk bisa menghasilkan karakter huruf kapital <em><strong>&ldquo;E&rdquo;</strong></em> sebagai awal addressnya nanti.
